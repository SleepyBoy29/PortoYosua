Hashicon — Identicon Generator

Hashicon is an open source project initially made for Ethereum Classic. I created Hashicon with the intention to improve the user experience of blockchain applications. This library can generate an icon from a hash like "0xA25727427...7B26A8051." Having a visual representation of a hash / cryptographic address helps to identify these hard-to-distinguish collections of letters and numbers, while also adding an emotional and decorative component. In this way, Hashicon additionally provides some extra security against fraud and user mistakes.
